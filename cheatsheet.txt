******* TMUX *******

Re size pane: C-Space <hold option and arrow key>


******* Neovim *******

Toggle trouble view: <leader>xx

spell suggestions: z=
jump to next misspelled word: ]s
jump to previous misspelled word: [s
add word to spell dictionary: zg

******* Pyright ******

pyright runs in single file mode by default. runs with
a root directory if it finds a .gitignore file, or 
any other indication file. needs to restart if new 
files are added and imported while LSP is running 
(otherwise it won't recognize them)
